# 🏢 Vidrieria-SAS - Sistema de Gestión Empresarial

Aplicación web orientada a la gestión de operaciones comerciales en una empresa de vidriería, enfocada en la organización de productos, clientes y procesos internos.

---

## 📌 Descripción

Vidrieria-SAS es un sistema desarrollado para digitalizar y organizar procesos comerciales dentro de una empresa del rubro de vidrios.

El proyecto permite estructurar información que normalmente se maneja de forma manual, mejorando el control, la organización y la eficiencia operativa.

---

## 🎯 Objetivo del proyecto

Desarrollar una solución que permita:

- gestionar productos y servicios  
- organizar información de clientes  
- mejorar el control de operaciones  
- digitalizar procesos comerciales  
- facilitar la administración del negocio  

---

## ❗ Problema que resuelve

En muchos negocios pequeños o medianos:

- los procesos se manejan manualmente  
- no existe un control claro de información  
- los datos están dispersos  
- no hay herramientas digitales adecuadas  

Este sistema permite centralizar y estructurar la información.

---

## ⚙️ Funcionalidades actuales

- 🏢 Gestión de productos (vidrios, servicios, etc.)  
- 👥 Registro de clientes  
- 📄 Organización de información comercial  
- 📊 Estructuración de datos  
- 🔎 Consulta de registros  

---

## 🏗️ Arquitectura del sistema

El sistema sigue una arquitectura básica de aplicación web:

- 🌐 Frontend → interfaz de usuario  
- ⚙️ Backend → lógica de negocio  
- 🗄️ Datos → almacenamiento en estructuras simples
